📋 Formato File Excel ⬇ Template

Il file .xlsx deve avere queste colonne nella prima riga (intestazione). Scarica il template per iniziare subito.

topic *
Argomento del video Instagram. Scrivi in italiano, chiaro e conciso.
es. “essiccazione naturale vs forno”
pilastro *
Categoria del contenuto. Valori accettati:
tecnico prodotto servizio brand
photoId
Nome del file foto nella cartella assets/photos/. Opzionale.
es. “faggio_premium.jpg”
note
Note libere per uso interno. Non vengono usate dalla pipeline.
es. “Da pubblicare in alta stagione”
parlato  — opzionale, bypassa Claude
Testo parlato completo già scritto da te. Se compilato, Claude non viene chiamato e questo testo viene passato direttamente a HeyGen. Se vuoto, Claude genera il testo automaticamente.
es. “La legna umida crea variazioni di temperatura nel forno. Ogni nostro lotto viene testato con igrometro… Se vuoi provare la nostra legna, contattaci.”
⏱ 80–120 parole · 30–45 secondi · termina sempre con la CTA fissa
status
Stato iniziale del topic. Se omesso → pending.
pending done skip
⚠ Note: I campi con * sono obbligatori. La prima riga deve essere l’intestazione con i nomi esatti delle colonne. Le righe con topic vuoto vengono ignorate. Caricare un nuovo file sovrascrive i topic precedenti.
